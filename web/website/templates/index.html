This page is an interactive client for the PictureServer JSON API defined in interface.md.
UPDATE: The server boots a real YOLOv8n‑cls classifier behind a strict JSON contract. Each /classifier success increments success and each malformed upload increments fail. Try it below — the box on the right logs every HTTP round‑trip in real time.

The protocol mandates one error envelope and exact response shapes. Tokens are JWT HS256 with persistent revocation; /logout actually invalidates the token. /status.uptime is fractional and monotonic. Scores are strictly in (0, 1] and sum to at most 1.
