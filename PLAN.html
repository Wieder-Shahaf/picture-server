Locked decisions

  • Framework: Flask (synchronous).
  • Classifier: Ultralytics YOLOv8n-cls; weights baked into the image at build; sentinel inference at boot sets health.
  • Persistence: SQLite in /app/data/app.db, WAL mode, bind-mounted to ./data/app.db; bcrypt password hashes; persistent JWT-revocation table.
  • Tokens: JWT HS256 with sub, jti, iat, exp=+24h. /logout inserts jti into revoked_tokens.
  • Port: 5000. Host: 0.0.0.0.
  • Imports: absolute inside website/; no from x import *.
  • Contract target: docker-compose up from repo root.

SPEC CORNERS (survival guide)

  1. Error envelope {"error":{"http_status":N,"message":"..."}} on every error path, with Content-Type: application/json, and N == response.status_code. Override Flask defaults for 400/401/404/405/500 + generic Exception.
  2. Bearer parsing: exact Bearer <token>; missing/malformed/expired/bad-signature/revoked → 401.
  3. JSON body: missing fields or non-JSON → 400 (use get_json(silent=True)).
  4. Wrong method → 405 with envelope (not HTML).
  5. No redirects (app.url_map.strict_slashes = False).
  6. /register 201 body exactly {"message":"User registered successfully"}.
  7. /logout 200 body exactly {"message":"Logged out successfully"}.
  8. /login 200 body contains token.
  9. /status body schema exact; uptime is fractional float (e.g. 230.7); api_version: 1.
  10. Counter rules: /classifier 200 → success++; /classifier 400 → fail++; /classifier 401 → no change; 5xx → no change; other endpoints never touch counters.
  11. Classifier response: {"matches":[{"name":str,"score":float}...]}; each score 0<score≤1; sum ≤ 1; drop zero scores.
  12. Persistence across container restarts: users + revoked tokens must survive.
  13. Thread safety on counters (threading.Lock) + SQLite WAL.
  14. Idempotent logout: re-using a revoked token → 401.
  15. No stack-trace leak on 500.

Our 5 interop tests (bug-magnets)

  • T1 — Error envelope http_status field matches HTTP code across 400/401/405/409.
  • T2 — Counter accuracy: 1 valid PNG → +1 success; 2 malformed → +2 fail.
  • T3 — /logout actually revokes: protected call with the same token after logout → 401.
  • T4 — /status.uptime is fractional and strictly grows by ≥ 1.0 s.
  • T5 — Classifier score invariants: each in (0,1], sum ≤ 1.

Each test is anchored to an explicit sentence in interface.md. No probing of unspecified behavior.
